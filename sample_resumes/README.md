# Sample Resumes

Place sample PDF or DOCX resumes in this directory for testing.

## Supported Formats
- `.pdf` — PDF documents
- `.docx` — Microsoft Word documents

## Notes
- The system extracts text from each resume and compares it against the job description.
- Image-based (scanned) PDFs are not supported — the resume must contain selectable text.
- Legacy `.doc` files are not supported — please convert to `.docx` first.
